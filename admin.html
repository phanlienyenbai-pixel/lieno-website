Kết nối GitHub để quản trị website

Nhập GitHub Personal Access Token của bạn để bắt đầu chỉnh sửa website. Token được lưu trong trình duyệt, không gửi đi đâu ngoài GitHub.

💡 Chưa có token? Vào github.com/settings/tokens/new → tick repo → Generate → copy token dán vào đây.

⚠️ Token chỉ hiện 1 lần duy nhất khi tạo. Nếu mất token, vào GitHub tạo lại.
